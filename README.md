## 🏅 Olympics Data Analysis Dashboard (Tableau)

This project presents an interactive Tableau dashboard analyzing historical Olympics data, focusing on athlete participation, performance, and medal distribution.

## 🚀 Overview

The dashboard explores trends in Olympic Games over time, including gender participation, athlete growth, physical attributes, and country-wise performance.

## 📊 Key Features
1.	👥 Male vs Female Participation - Tracks gender participation trends across Olympic years
2.	📈 Athlete Growth Over Time - Shows total athletes participating by year and season (Summer/Winter)
3.	⚖️ Height vs Weight Correlation - Analyzes relationship between athletes' physical attributes
4.	🌍 Top Countries by Medals - Highlights countries with highest medal counts
5.	🏅 Top Sports by Medals - Identifies sports contributing most to total medals
6.	📊 Distribution Analysis - Visualizes age, height, and weight distribution of medal winners


## 🛠️ Tools & Technologies
1.	Tableau
2.	Data Cleaning & Preparation
3.	Data Visualization & Dashboard Design


## 🎯 Insights
1.	Male participation has historically been higher, but female participation is increasing steadily
2.	Athlete participation has grown significantly over time, especially in Summer Olympics
3.	Strong positive correlation between height and weight
4.	USA leads in total medal count
5.	Athletics and Swimming dominate medal distribution
