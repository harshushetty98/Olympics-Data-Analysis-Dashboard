## 🏅 Olympics Data Analysis Dashboard (Tableau)

This project presents an interactive Tableau dashboard analyzing historical Olympics data, focusing on athlete participation, performance, and medal distribution.

## 🚀 Overview

The dashboard explores trends in Olympic Games over time, including gender participation, athlete growth, physical attributes, and country-wise performance.

## 📊 Key Features
👥 Male vs Female Participation
Tracks gender participation trends across Olympic years
📈 Athlete Growth Over Time
Shows total athletes participating by year and season (Summer/Winter)
⚖️ Height vs Weight Correlation
Analyzes relationship between athletes' physical attributes
🌍 Top Countries by Medals
Highlights countries with highest medal counts
🏅 Top Sports by Medals
Identifies sports contributing most to total medals
📊 Distribution Analysis
Visualizes age, height, and weight distribution of medal winners


## 🛠️ Tools & Technologies
Tableau
Data Cleaning & Preparation
Data Visualization & Dashboard Design


## 🎯 Insights
Male participation has historically been higher, but female participation is increasing steadily
Athlete participation has grown significantly over time, especially in Summer Olympics
Strong positive correlation between height and weight
USA leads in total medal count
Athletics and Swimming dominate medal distribution
